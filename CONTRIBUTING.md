# Contributing to Whispr

Welcome! We appreciate your interest in making Whispr more secure and accessible.

## 🛠️ Development Setup

### System Requirements
- Node.js (v18+)
- npm or yarn
- Git

### Initializing the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/utksh1/Whispr.git
   cd Whispr
   ```

2. Setup the Client:
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. Setup the Server:
   ```bash
   cd ../server
   npm install
   node index.js
   ```

## 🔐 Security Disclosures
If you find a security vulnerability, please do **not** open a public issue. Instead, contact the maintainers directly.

## 🌿 Branching Strategy
- `main`: Production-ready code and documentation.
- `feat/`: New features.
- `fix/`: Bug fixes.
- `security/`: Security-related updates.

## 🤝 Code of Conduct
Please be respectful and constructive in all interactions.
